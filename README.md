# DickTrainer

## Keyboard Controls

### Graph Screen

| Key | Action |
|---|---|
| `Enter` | Start graph / advance to next customer |
| `Space` | Pause / Resume |

### Volume Control

| Key | Action |
|---|---|
| `Home` | Volume +5 (max 100) |
| `End` | Volume -5 (min 0) |

### Transparency Control

| Key | Action |
|---|---|
| `Page Up` | Transparency +5% (max 100%) |
| `Page Down` | Transparency -5% (min 0%) |

### Position Control

| Key | Action |
|---|---|
| `↑` | Align graph to top |
| `↓` | Align graph to bottom |
| `←` | Move graph left (right → center → left) |
| `→` | Move graph right (left → center → right) |
